🔔 Configuration des alertes email

Comment configurer EmailJS (gratuit) :
1. Créer un compte sur emailjs.com
2. Ajouter un service email (Gmail, Outlook…) → noter le Service ID
3. Créer un template avec les variables {{action}}, {{variation}}, {{score}}, {{message}} → noter le Template ID
4. Copier votre Public Key depuis Account → API Keys

Comment configurer Twelve Data (cours live) :
1. Créer un compte gratuit sur twelvedata.com (800 req/jour, pas de CB)
2. Copier votre API Key depuis le Dashboard → API Keys
3. Coller dans le champ « Clé Twelve Data » ci-dessus et sauvegarder
4. Rechargez la page — les cours s'afficheront automatiquement

Comment configurer GNews (actualités) :
1. Créer un compte gratuit sur gnews.io (100 requêtes/jour)
2. Copier la API Key depuis votre tableau de bord
3. La coller dans le champ « Clé GNews API » ci-dessus et sauvegarder
