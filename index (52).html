🔔 Configuration des alertes email

Comment configurer EmailJS (gratuit) :
1. Créer un compte sur emailjs.com
2. Ajouter un service email (Gmail, Outlook…) → noter le Service ID
3. Créer un template avec les variables {{action}}, {{variation}}, {{score}}, {{message}} → noter le Template ID
4. Copier votre Public Key depuis Account → API Keys
5. Mettre à jour le fichier stocks.json avec ces valeurs dans la section config (optionnel)
